2) How to embed audio and video in a webpage? 

- The HTML <audio> element is used to embed an audio file on a web page.

- The HTML <video> element is used to embed video file on a web page.

3)  Semantic element in HTML5?

4)  Canvas and SVG tags?

- Canvas tag is used to draw graphics through javascript.

- SVG tag defines the container for SVG graphics.
